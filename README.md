# Pokemon Battler

A C++ Pokemon battle simulator that implements core Pokemon game mechanics including Pokemon, Moves, Trainers, and Battles.

## Features

- **Pokemon System**: Create Pokemon with custom attributes (name, HP, type, move)
- **Move System**: Define moves with name, power, and type
- **Trainer System**: Manage up to 6 Pokemon in a trainer's belt
- **Battle System**: Simulate turn-based battles between Pokemon
- **Damage System**: Pokemon can take damage and faint

## Running the Program
After building, run the executable:

bash
./C__
